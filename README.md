# 🎓 Student Career Prediction using ML & Deep Learning

This project predicts the future career path of Computer Science students using Machine Learning and Deep Learning models.

Models Used:
- KNN (K-Nearest Neighbors)
- ANN (Artificial Neural Network)
- CNN (Convolutional Neural Network)

The application is deployed using Streamlit.

---------------------------------------------------

Project Structure

student-career-prediction/
│
├── app.py

---------------------------------------------------

Installation

1. Clone the repository

git clone https://github.com/yourusername/student-career-prediction.git

2. Install requirements

pip install -r requirements.txt

3. Run the Streamlit App

streamlit run app.py

---------------------------------------------------

Input Features

- Age
- Gender
- GPA
- Programming Skill
- Algorithm Skill
- Mathematics Skill
- Communication Skill
- Teamwork
- Leadership

---------------------------------------------------

Output

The system predicts the student's future career category using three models:

- KNN
- ANN
- CNN
